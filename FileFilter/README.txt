## Требования
Требуемая версия Java для запуска программы: Java 8 или выше.

## Пример команды для запуска программы
Прежде чем выполнить команду для запуска программы, необходимо создать JAR-файл проекта. Файлы, которые нужно отфильтровать, должны находится в одной папке с JAR-файлом.
Затем откройте консоль и перейдите в папку, содержащую JAR-файл. После этого выполните команду для запуска программы. Пример команды:
java -jar FileFilter.jar -o "C:\Users\kural\IdeaProjects\OopCourse\FileFilter\results" -p "sample-" -s file1.txt file2.txt file3.txt

## Особенности реализации
При фильтрации данных по типам целых и вещественных чисел, следует учитывать, что целые числа также являются подмножеством вещественных чисел.
В данной программе целые числа будут отображены как отдельная категория, не включенная в список вещественных чисел.

Для целых чисел использован тип BigInteger вместо Integer, а для вещественных чисел - тип double вместо float.
Это было сделано для того, чтобы полностью соответствовать приведенному в задании примеру и обеспечить более точное представление данных.
